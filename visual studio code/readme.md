## 中文语言包安装

1、command + shift + P 调出命令窗口

2、输入 Configure Display Language ，安装简体中文扩展

## 创建 vue 代码片段

1、先安装 Vetur 扩展 让 VSCode 支持 .vue 的扩展名

2、然后 菜单栏 -文件 - 首选项 - 用户代码片段

3、输入 vue 点击进行设置
